# Sweet Juice Plugins

This directory contains standard plugins for the Sweet Juice framework.

## Available Plugins

| Plugin | Status | Description |
|--------|--------|-------------|
| [**biometric**](./biometric/README.md) | ✅ | Fingerprint and Face ID authentication. |
| [**daemon**](./daemon/README.md) | ✅ | Keeps Go active via Android Foreground Services. |
| [**datadir**](./datadir/README.md) | ✅ | Access standard app directories (files, cache). |
| [**datastore**](./datastore/README.md) | ✅ | Simple key-value store using SharedPreferences. |
| [**deeplinking**](./deeplinking/README.md) | ✅ | Handle incoming app links and custom schemes. |
| [**devicestate**](./devicestate/README.md) | ✅ | Monitor battery, connectivity, and power mode. |
| [**filepicker**](./filepicker/README.md) | ✅ | Select files and media from the device. |
| [**logger**](./logger/README.md) | ✅ | Route Go logs to native platform loggers (Logcat). |
| [**notification**](./notification/README.md) | ✅ | Post and manage system notifications. |
| [**osapi**](./osapi/README.md) | ✅ | Get OS version and device hardware information. |
| [**permission**](./permission/README.md) | ✅ | Manage runtime system permissions. |
| [**workmanager**](./workmanager/README.md) | ✅ | Schedule background tasks that run even if the app is closed. |
